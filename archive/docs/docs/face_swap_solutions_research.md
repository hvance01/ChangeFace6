## 二、各方案详细分析

### 1. Akool (国际)

**官网**: https://akool.com/apps/faceswap

| 项目 | 详情 |
|------|------|
| **效果质量** | ⭐⭐⭐⭐⭐ 业界领先，支持4K |
| **API类型** | RESTful API + Webhook |
| **定价模式** | Credits积分制 |
| **视频换脸价格** | 10 credits / 10秒 (1080p: 5 credits/10秒) |
| **免费额度** | 100 credits (约2个视频) |
| **订阅计划** | Pro/Pro Max/Business/Enterprise |
| **最大视频长度** | Business级别更长 |

**优点:**
- 效果最好，4K分辨率支持
- 企业级客户(Canon、Google Cloud、Coca-Cola)验证
- 完善的API文档和Postman支持
- 支持实时换脸(Live Face Swap)

**缺点:**
- 价格较高
- 需要翻墙访问
- 企业定价需单独询价
- 非中国本土服务，合规风险

---

### 2. DeepSwap (国际)

**官网**: https://www.deepswap.ai/

| 项目 | 详情 |
|------|------|
| **效果质量** | ⭐⭐⭐⭐ Deepfake级别 |
| **API类型** | RESTful API + SDKs (Python/Node/Java/Go) |
| **定价模式** | Credits积分制 |
| **订阅价格** | $19.99/月 (20 credits) 或 $49.99/年 |
| **视频限制** | 免费480p，付费更高 |
| **SLA** | 99.9% 可用性 |

**优点:**
- 自研模型，换脸相似度>90%（比开源高20%）
- 支持同时换6张脸
- 完善的SDK支持
- 批量处理能力
- 文件自动加密清除

**缺点:**
- API为付费功能
- 需要翻墙访问
- 免费版分辨率低(480p)
- 视频长度限制(付费2分钟)

### 3. 腾讯云人脸融合 (中国)

**官网**: https://cloud.tencent.com/product/facefusion

| 项目 | 详情 |
|------|------|
| **效果质量** | ⭐⭐⭐⭐ 人民日报军装照验证 |
| **API类型** | REST API + SDK (Java/Python/Go/Node/PHP/C#) |
| **定价模式** | 调用次数/QPS/并发任务数 |
| **免费额度** | 每活动500次免费调用 + 10个素材额度 |
| **视频版本** | 短视频版/长视频基础版/长视频专业版/影视级离线 |
| **QPS限制** | 默认20 QPS，可购买至300 |

**优点:**
- 国内合规，无需翻墙
- 支持6人脸指定融合
- 多种视频版本选择
- 官方SDK完善
- 毫秒级响应(图片)，视频≤1/2时长
- 经过大型活动验证(人民日报军装照)

**缺点:**
- 会添加水印
- 需要购买素材额度
- 视频版本价格需单独询价
- 公众人物/敏感内容限制
- 可能需要ICP备案

### 4. 商汤科技 SenseAR (中国)

**官网**: https://openar.sensetime.com/

| 项目 | 详情 |
|------|------|
| **效果质量** | ⭐⭐⭐⭐⭐ 技术领先 |
| **API类型** | SDK (C/Java/Unity) |
| **定价模式** | 企业定制 |
| **主要功能** | AR特效、美颜、人脸跟踪 |

**⚠️ 重要发现:**
商汤科技的SenseAR平台**主要面向B端企业客户**，且其"换脸"功能主要是**检测和防范换脸攻击**的安全功能，而非提供娱乐性的人脸融合API。

**优点:**
- 技术实力顶级
- 完整的AR/人脸技术栈
- 跨平台SDK支持
- 适合深度定制

**缺点:**
- **不直接提供视频换脸API服务**
- 仅限企业客户
- 需要商务对接
- 接入周期长
- 价格不透明

**结论: ❌ 不满足需求** - 商汤的换脸技术主要是安全检测，非视频换脸生成

---

### 5. 阿里云视觉智能 (中国)

**官网**: https://vision.aliyun.com

| 项目 | 详情 |
|------|------|
| **效果质量** | ⭐⭐⭐⭐ |
| **API类型** | REST API + SDK (Java/Python/Go/Node/PHP/C#) |
| **API名称** | MergeVideoFace / MergeVideoModelFace |
| **定价模式** | 按量付费/预付费资源包/预付费QPS |
| **计费单位** | 按视频输出时长（秒）计费 |
| **最小计量** | 1秒 |

**两种调用方式:**
1. **通用视频人脸融合** - 上传视频+人脸图片即可
2. **模板视频人脸融合** - 预先创建模板，效率更高

**优点:**
- 国内合规，无需翻墙
- 异步调用，失败不计费
- 支持多人脸指定换脸
- 2024年7月后开通可直接使用
- 资源包可跨能力使用

**缺点:**
- 会添加"Generated by AI"水印
- 禁止公众人物/敏感内容
- 具体价格需查询官方文档
- 需要OSS存储配合

### 6. 火山引擎 (中国-字节跳动)

| 项目 | 详情 |

- 人像融合主要是**图片级别**

## 四、推荐建议

### 场景A: 面向国内用户的营销视频
**首选: 腾讯云人脸融合** 或 **阿里云视觉智能**
- 合规性好，无需翻墙
- SDK完善，集成简单
- 效果经过大量验证

### 场景B: 追求最佳效果，不考虑国内合规
**首选: Akool**
- 4K支持，效果最佳
- 企业级验证

### 场景C: 成本敏感，中等效果即可
**首选: DeepSwap** (国际) 或 **阿里云** (国内)
- 价格相对透明
- 批量处理支持好

### ❌ 不推荐: 商汤科技
- 不提供直接的视频换脸API
- 主要是B端企业定制
- 换脸功能是安全检测而非生成

### ⚠️ 待确认: 火山引擎
- 人像融合主要是图片级别
- 视频换脸需要用即梦AI数字人方案
- 功能定位可能不完全匹配需求

## 六、当前项目状态

当前ChangeFace项目使用的是 `okaris/roop` (Replicate)，配置如下:

```python
# config.py
FACE_SWAP_MODEL = "okaris_roop"
API_CONFIGS = {
    "okaris_roop": {
        "model": "okaris/roop:8c1e100ecabb3151cf1e6c62879b6de7a4b84602de464ed249b6cff0b86211d8",
        "cost": 0.089  # USD (~$0.089 per run)
    }
}
```

### 升级建议

1. **短期**: 在现有Replicate基础上添加 `enhance_face: True` 参数
2. **中期**: 集成腾讯云或阿里云作为国内备选
3. **长期**: 根据业务需求选择Akool(效果优先)或自托管FaceFusion(成本优先)
