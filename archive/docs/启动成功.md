## 📝 使用步骤

### 1. 打开浏览器
在浏览器中访问 http://localhost:8501

### 2. 上传文件
- **左侧**: 上传头像照片（要替换成的脸）
  - 格式: JPG, PNG
  - 大小: < 10MB
  - 建议: 清晰的正面照片

- **左侧下方**: 上传营销视频
  - 格式: MP4, MOV
  - 大小: < 500MB
  - 时长: 建议 < 60秒

### 3. 开始处理
- 点击右侧的 **"🚀 开始换脸"** 按钮
- 等待处理（约1-2分钟）
- 进度条会显示处理状态

### 4. 下载结果
- 处理完成后，视频会自动播放
- 右键视频 → 另存为
- 或点击下载链接

## ⚠️ 常见问题

### Q: 无法访问应用？
A: 检查：
1. 应用是否在运行: `ps aux | grep streamlit`
2. 端口是否被占用: `lsof -i :8501`
3. 防火墙是否开放 8501 端口

### Q: API Token 错误？
A:
1. 检查 .env 文件中的 token 是否正确
2. 访问 https://replicate.com/account/api-tokens 重新获取
3. 重启应用使配置生效

### Q: 上传失败？
A: 检查：
1. 文件格式是否支持
2. 文件大小是否超限
3. 网络连接是否正常

### Q: 处理很慢？
A:
- 正常处理时间为 1-2 分钟
- 视频越大处理越慢
- 建议先用小视频测试
